Environment Setup =============================================================================== Assignments in this course will be done in either C++ or Rust. The following programs must be installed on your personal machine: - Git: version control, used to retrieve and synchronize the starter code - CMake: enables cross-platform compilation - Recent C++ compiler: must support at least C++17 This first assignment describes the setup steps for different platforms (Windows, Linux, macOS). For Rust, I will only describe its installation on Windows and Visual Studio Code. Visual Studio Code (all platforms / Rust) ------------------------------------------------------------------------------ To install Rust on your machine, use the following site: https://rustup.rs/. This site will ask you to run a command or launch a local installer. This will install the latest stable version of Rust. Next, install Visual Studio Code: https://code.visualstudio.com/. Finally, launch Visual Studio Code with your project and install the following extension: `rust-analyzer`. Note that during installation or at restart, Visual Studio Code may ask you to install additional components for Rust (via Cargo). Accept them. ![VS Code extension installation - Rust](imgs/d0-code-ext-rust.png) To compile Assignment 0, open a terminal and run the following command in the project folder: ``` cargo run --release --example=devoir0 -- ``` All dependencies as well as the starter code will compile automatically. Assignment =============================================================================== In this assignment, we will get familiar with some concepts covered in class related to colors and linear algebra. Open the file corresponding to the assignment: `examples/devoir0.rs`. You will edit this file directly. Do not be intimidated by its size: it contains many comments. Read them carefully, as they provide instructions on how to complete the different tasks. You can use the following command to recompile and run the code: ``` cargo run --release --example=devoir0 -- ``` Note that the working directory is the folder where `cargo` is executed. ## Task 1: Linear Algebra Go to the function `task_1_maths()` and fix all the __TODO__ items as well as any errors displayed during execution. ## Task 2: Colors and Images Go to the function `task_2_images(...)` and fix all the __TODO__ items as well as any errors displayed during execution.
